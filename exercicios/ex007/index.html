Principais formatações de texto em HTML5

Negrito/destaque

Nesta frase temos termo em negrito usando a tag <b>(Não Semântica)

Nesta frase temos termo em negrito usando a tag <strong> (Semântica)

Itálico / Ênfase

Nesta frase temos termo em itálico usando a tag <i> (Não Semântica)

Nesta frase temos termo em Itálico usando a tag <em> (Semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>

Podemos criar também um texto marcado usando a tag <mark>

Texto grande e pequeno

Estamos criando um Texto grande e um texto pequeno nesse parágrafo usando as tags <big> e <small> respectivamente

Texto deletado

Podemos marcar um texto como excluído usando a tag <del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois usando a tag <ins>

Existe também o sublinhado com a tag <u> (não semântica)

Texto sobrescrito

Para inserir coisas do tipo x20+3 usamos a tag <sup>

Texto subscrito

Para inserir coisas do tipo H2O usamos a tag <sub>
